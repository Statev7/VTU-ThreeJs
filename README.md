# VTU-ThreeJs

<h2>How to start the project</h2>

	npm install
	npm run dev

<h2>Summary</h2>
<p>This application uses the Three.js library to create a 3D scene. The main element of the scene is a cube, and upon clicking on the cube, it shatters into smaller cubes. </p>
